# Feedback

Thank you for using 2LY ! Your feedback is crucial for improving the platform.

## How to Provide Feedback

### Bug Reports
- Use our [GitHub Issues](https://github.com/2ly/2ly/issues) to report bugs
- Include steps to reproduce, expected vs actual behavior
- Attach logs and screenshots when relevant

### Feature Requests
- Submit feature requests through GitHub Issues
- Describe the use case and expected functionality
- Include examples of how the feature would be used

### General Feedback
- Join our community discussions
- Share your experience and suggestions
- Help other users with questions

## What We're Looking For

- Performance and stability issues
- UI/UX improvements
- Missing features or functionality
- Integration challenges
- Documentation gaps

## Beta Program Guidelines

- Beta features may change or be removed
- Regular updates and breaking changes expected
- Early access to new features
- Direct communication with development team

## Contact

For urgent issues or private feedback, contact us at dev@2ly.ai
