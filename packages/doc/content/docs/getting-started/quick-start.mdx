Create a workspace, connect an agent, and configure your first tool.

## Prerequisites

[Install 2LY](/getting-started/installation) and ensure it's running at http://localhost:8888

## 1. Create Workspace

1. Open http://localhost:8888
2. Create a new workspace

[add screenshot]

## 2. Install an MCP Server

2LY comes with a selection of pre-installed server. Use one of these to quickly grasp how 2LY works. You'll have the chance to [add any MCP Servers](links-to-add-mcp-server-doc) later.

[add the doc to explain how to add any mcp server]

1. Click **Browse MCP Servers**
2. Select the **CHOOSE AN INTERESTING SERVER**
3. Click **Test Server**
4. Click **Save**

[add screenshot, complete the steps]

## 3. Create a Toolset

Toolsets allow you to shortlist and combines which tools will be advertised to which agents.

1. Click **Create Tool Set**
2. Give it a meaningul name such as **News summarizer**
3. Select the tools to include (add at least one)

## 4. Connect 2LY to your agent

Your toolset is now ready to be consumed by your agents.

1. Click **Connect**
2. Select the type of agent you would like to connect. If unsure use the JSON configuration
3. Launch your agent with 2LY connected

Et voilà !

## 5. Test in Playground (Optional)

1. Choose a tool
2. Add instruction and click **Test**
3. Verify the execution of the interaction

## Verification

You should have:

- ✅ Workspace created
- ✅ Sandbox agent running
- ✅ Configured tool connected

## Troubleshooting

**Can't access localhost:8888?**

```bash
npm run start
docker-compose ps
```

**MCP test fails?**

- Check directory permissions
- Try `/tmp` instead of Desktop
