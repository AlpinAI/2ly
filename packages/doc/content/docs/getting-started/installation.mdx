Installing 2LY takes 2 minutes.

## Prerequisites

- Node.js 22+
- [Docker](https://docs.docker.com/)

## Installation

```bash
git clone https://github.com/AlpinAI/2ly.git
cd 2ly
npm run start # calls docker compose
```

Access the dashboard at **http://localhost:8888**, create your workspace, and follow our 3-step onboarding (takes 2 minutes).

## Runtime Only

Install just the runtime package:

```bash
# Global install
npm install -g @2ly/runtime

# Or use directly
npx @2ly/runtime
```

## Python SDK

For Python agents:

```bash
pip install twoly
```

## Troubleshooting

**Port conflicts:** Ports 8888, 3000, 4222 must be available

**Docker not running:**

```bash
docker --version
docker-compose --version
```

**NATS connection failed:**

```bash
docker logs 2ly_nats
```
