Get 2LY running on your local machine.

## Prerequisites

- Node.js 22.x or higher
- Docker and Docker Compose
- Git

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/AlpinAI/2ly.git
cd 2ly
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Services

```bash
npm run start
```

This starts:
- Backend (GraphQL API)
- Frontend (React UI)
- DGraph (database)
- NATS (message broker)

### 4. Access Application

- **Frontend**: http://localhost:8888
- **GraphQL API**: http://localhost:4000/graphql

## Runtime Only

Install just the runtime package:

```bash
# Global install
npm install -g @2ly/runtime

# Or use directly
npx @2ly/runtime
```

## Python SDK

For Python agents:

```bash
pip install twoly
```

## Troubleshooting

**Port conflicts:** Ports 8888, 4000, 4222 must be available

**Docker not running:**
```bash
docker --version
docker-compose --version
```

**NATS connection failed:**
```bash
docker logs 2ly_nats
```
