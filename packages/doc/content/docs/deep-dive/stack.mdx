2LY comes with several technical components providing a unified setup

# Docker images

- **backend** - typically exposed on HTTP 3000
- **frontend** - typeically exposed on HTTP 8888
- **nats** - message broker, typically exposed on TCP 4222
- **dgraph** - graph database

# Local setup

The repository comes with a `docker-compose.yml` file ready to be started with reasonable configuration.
